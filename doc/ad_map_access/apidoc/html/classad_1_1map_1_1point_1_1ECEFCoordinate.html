Detailed Description

DataType ECEFCoordinate.

equatorial WGS-84 earth radius [m]= 6378137 Height of the Mount Everest [m]= 8848; Maximum ECEF Coordinate: WGS84_R + MOUNT_EVEREST ~ 6400000 The unit is: Meter
